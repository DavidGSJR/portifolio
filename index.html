Experiencia Profissional

Sistema salão de beleza

O presente projeto tem por objetivo apresentar um sistema para gerenciamento de serviços para o Studio de beleza Realce Cabeleireiros. O sistema foi desenvolvido para o controle de agendamento de horários, cadastro de clientes e funcionários e gerenciamento de serviços assim como o controle financeiro (fluxo de caixa) simplificado, visando o controle do Studio de beleza de maneira simples, trazendo ferramentas que facilitam o trabalho diário. O sistema será interno e monitorado pelos funcionários e o gerente do Realce Cabeleireiros, onde cada usuário terá suas funções diferentes no sistema sendo necessário um sistema de Login para cada um desempenhar suas funções dentro do sistema. O banco de dados foi criado para armazenar as informações dos usuários, clientes e serviços da empresa. As ferramentas utilizadas foram Java, JavaScript, JSP, CSS, Servelet, MySQL e JQuery.
